# JM Tally

Indian SME–oriented ledger: transactions, sales invoices, GST helpers, bank import, multi-company workspaces. Built with **React** and **Vite**.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Optional: set `VITE_ANTHROPIC_API_KEY` in `.env` for live AI assistant replies (see `.env.example`).
